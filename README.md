Defiled Lands

A Minecraft 1.20.1 Forge mod that adds a dark, corrupted dimension — the Defiled Lands.Ported from the 1.12.2 version of Defiled Lands (with permission from the original author).

Features

Defiled Biomes — Unique biomes with corrupted terrain, flora, and atmosphere

New Creatures — Custom mobs native to the defiled dimension

Blocks & Items — Defiled variants of blocks, tools, weapons, and materials

Corruption System — Spreading corruption mechanics

Dungeon Generation — Procedurally generated dungeons with bosses and loot

World Generation — TerraBlender-based biome integration

Improvements Over the 1.12.2 Version

This version introduces numerous configurable options, allowing players to freely adjust their gameplay experience:

Corruption Spread Mechanics

Spread Toggle: Choose whether defiled blocks automatically spread corruption (disabled by default)

Area Restriction: Option to limit corruption spread to defiled biomes only

Speed & Range: Freely adjust the number of spread attempts per random tick (1-20) and the spread radius per attempt (1-8 blocks)

World Generation

Biome Weight: Control the generation frequency of defiled biomes via TerraBlender region weight (0-50); set to 0 to completely disable

Dungeon Generation: Choose whether defiled dungeons generate underground in defiled biomes

Book Wyrm Breeding & Output

Luck Influence: Set the luck attribute multiplier for golden Book Wyrm breeding chances (0.0-10.0). Formula: actual chance = base chance × (1 + luck × multiplier)

Level Cap: Adjust the maximum level Book Wyrms can achieve through breeding (1-10000), affecting their book digestion capacity

Enchantment Filtering:

Supports blacklist/whitelist mode switching

Customizable enchantment ID list for precise control over which enchantments Book Wyrms can produce
