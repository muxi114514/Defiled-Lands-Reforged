# Defiled Lands

A Minecraft 1.20.1 Forge mod that adds a dark, corrupted dimension — the Defiled Lands.

## Features

- **Defiled Biomes** — Unique biomes with corrupted terrain, flora, and atmosphere
- **New Creatures** — Custom mobs native to the defiled dimension
- **Blocks & Items** — Defiled variants of blocks, tools, weapons, and materials
- **Corruption System** — Spreading corruption mechanics
- **Dungeon Generation** — Procedurally generated dungeons with bosses and loot
- **World Generation** — TerraBlender-based biome integration

## Requirements

| Dependency | Version | Required |
|---|---|---|
| Minecraft Forge | 47.x (1.20.1) | ✅ Yes |
| TerraBlender | 3.0.1+ | ✅ Yes |
| GlitchCore | 0.0.1+ | ✅ Yes (TerraBlender dependency) |
| Curios API | 5.x+ | ✅ Yes |
| JEI | 15.x+ | ❌ Optional |

## Installation

1. Install [Minecraft Forge](https://files.minecraftforge.net/) for 1.20.1
2. Install [TerraBlender](https://www.curseforge.com/minecraft/mc-mods/terrablender) and GlitchCore
3. Place the mod JAR into your `mods` folder
4. (Optional) Install Curios API for accessory slots
